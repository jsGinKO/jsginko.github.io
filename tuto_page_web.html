Comment créer un petit site web pour présenter vos projets

Que ce soit lors de votre cours de R3 ou autre, vous avez créer des projets sous forme d’une page html. Et maintenant vous souhaitez que le grand public puisse facilement accéder à ces documents.

pour cela je vous propose de créer un petit site web static via Github Pages.

Qu’est-ce que Github Pages

  • Github
  • Github Pages
    • site web statique hébergé dans un dépôt Github

Il faudra en plus faire appel à une technologie de génération de site web (Jekyll, Hugo, etc.)

Démarrage

  • création d’un compte Github
  • activation du service Github Pages

Création d’un compte Github

C’est par ici : https://github.com/signup?ref_cta=Sign+up&ref_loc=header+logged+out&ref_page=%2F&source=header-home Laissez-vous guider.

Attention, votre site aura pour adresse votre_nom_dutilisateur.github.io.

Création d’un dossier

Ce dossier hébergera le code de votre site et lui donnera aussi son nom.

  • cliquez sur le + en haut
  • choisissez New repository
  • nommez le votre_nom_dutilisateur.github.io c’est obligatoire pour que Github Pages fonctionne
  • assurez vous de bien le laissez en public
  • cliquez sur create repository

Création de la page html d’index

Dans l’encadré, quick setup, cliquez sur create a new file.

Nommez votre fichier obligatoirement index.html (à la place de *Name your file…).

Dans la zone d’édition, vous pouvez y mettre du code html. : par exemple

<h1> Bonjour internet !<h1>

Enregistrez cela en cliquant sur le bouton (vert) Commit changes.... Github vous propose de mettre un Commit message. Ce message est censé refléter la modification que vous entérinez en enregistrant vos modifications : comme ici : create index.html. Validez en cliquant sur Commit changes.

Maintenant votre dossier contient le fichier index.html.

Admirer votre page web

  • Au bout d’une minute, Github aura généré votre site web et vous devriez admirer votre page internet digne du début des années 2000 à l’adresse votre_nom_dutilisation.github.io.

Ajout de pages html

Il est possible de créer de vrais et beaux sites web (statique) hébergés par Github en utilisant des générateurs de sites statiques comme Jekyll ou Hugo.

Mais là je vous propose de faire au plus simple.

Il suffit de glisser déposer votre fichier html de votre ordinateur vers votre dépôt Github et votre fichier sera accessible à l’adresse votre_nom_dutilisateur.github.io/nom_de_votre_fichier.html.

Vous pouvez aussi ajouter dans votre fichier index.html un lien vers ce fichier avec la syntaxe :

<a href="https://votre_nom_dutilisateur.github.io/nom_de_votre_fichier.html">Ma super étude !</a>

Partager le lien vers votre page

Vous n’avez plus qu’à partager les liens de type votre_nom_dutilisateur.github.io/nom_de_votre_fichier.html via votre CV, votre lettre de motivation, vos réseaux sociaux, etc.

Vous pouvez éventuellement utiliser des raccourcisseur de lien ou des QR code pour les documents qui ont pour objectif d’être imprimé.
